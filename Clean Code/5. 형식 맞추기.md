의사소통의 일환
### 1. 적절한 행 길이 유지
- 신문 기사처럼 작성하라
  - 표제 - 첫 문단(커다란 그림) - 세부사항
  - 함수 이름 - 고차원 개념, 알고리즘 - 저차원 함수와 세부 내역
- 개념은 빈 행으로 분리하라
- 세로 밀집도: 연관성
- 수직 거리: 밀접한 개념은 세로로 가까이 둬야 함
  - 변수 선언: 변수는 사용한느 위치에 최대한 가까이 선언
  - 인스턴스 변수: 클래스의 맨앞
  - 종속 함수: 세로로 가까이 배치, 호출되는 순서로 정렬  
  - 개념적 유사
  - 호출하는 함수 - 호출되는 함수
  - 가장 중요한 개념 먼저 표현
### 2. 가로 형식 맞추기
- 가로 공백과 밀집도: 요소 나눔, 연산자 우선순위
- 가로 정렬
- 들여쓰기
